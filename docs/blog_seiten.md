# Website mit Blog und Seiten

## Blog auf der Hauptseite

Standardmässig ist Bludit für die Verwendung für einen Blog eingerichtet.

Die Inhalte können erstellt und direkt gespeichert werden (dies entspricht unter "Einstellungen" > "Erweitert" > "Art" der Auswahl "Standardvorgabe").

Zusätzlich zum Blog können Seiten eingerichtet werden, bespeischielsweise mit Informationen über den Blog, den Autor oder die Autorin. Diese werden mit der Auswahl "Statisch" gespeichert.

## Blog auf einer Seite

Soll der Blog nicht auf der Hauptseite ausgegeben werden, kann Bludit wie folgt eingerichtet werden:

1. Einrichtung einer statischen Seite als Hauptseite unter "Einstellungen" > "Allgemein" im Reiter "Erweitert" unter "Zugewiesene Seiten" > "Hauptseite" (damit wird auf derselben Seite der URL-Filter `/blog/` aktiviert).
2. Erstellung einer statischen Seite mit dem Titel "Blog".
