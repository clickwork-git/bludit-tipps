# Favicon

Favicon bezeichnet das kleine Symbol, Logo oder Bild, das in Browser-Reiter vor der Bezeichnung einer Seite gezeigt wird.

Das Favicon ist normalerweise eine Bilddatei mit den Massen 16 x 16 Pixel oder 32 x 32 Pixel.

Um einfach Favicons zu erstellen, kann ein Favicon-Generator verwendet werden.

## Favicon bei Bludit

Bludit zeigt standardmässig das Favicon mit dem Bludit-Logo.

In den meisten Fällen wird das Favicon des Themes verwendet. Diese kann mit einem eigenen Favicon ersetzt werden.

Bei den Themes Alternative und Blog X befindet sich die Datei des Favicons `favicon.png` im Verzeichnis `/bl-themes/alternative`beziehungsweise `/bl-themes/blogx/img`.
