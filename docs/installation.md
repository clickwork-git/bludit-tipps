Die Installation von Bludit ist einfach:

1. Die neueste Version von Bludit von der <a href="https://www.bludit.com" target="_blank">offfiziellen Website</a> herunterladen.
2.  Das ZIP-Archiv aud dem eigenen Computer entpacken.
3. Den entpackten Inhalt des ZIP-Archivs vom eigenen Computer auf den Server laden.

 Bludit kann im Hauptverzeichnis (root) oder in einem Unterverzeichnis wie beispielsweise `/bludit` installiert werden. Weitere Information dazu sind unter [Installationsverzeichnis](/installation/installationsverzeichnis) zu finden.

 Um den Inhalt des ZIP-Archivs auf den Server zu laden, kann ein FTP- oder SFTP-Client oder WebFTP verwendet werden. Weitere Informationen dazu sind unter [File Transfer (FTP und SFTP)](	/installation/file-transfer-ftp-und-sftp) zu finden.

4. Im Browser die Domain mit der Installation aufrufen.

 Befindet sich Bludit im Hauptverzeichnis unter https://meinedomain.ch, wenn es sich in einem Unterverzeichnis befindet beispielsweise unter https://meinedomain.ch/bludit.

5. Der Bludit-Installer führt darauf durch die Einrichtung der Website.

Für andere Installationsarten vgl. in der englischsprachigen Dokumentation [Easy Raspberry Pi Install](https://docs.bludit.com/en/how-to/easy-raspberry-pi-install) und [Installation on GNU/Linux](https://docs.bludit.com/en/how-to/installation-on-gnu-linux).
