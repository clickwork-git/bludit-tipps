# Update von Bludit v1.6 auf v2.0

Für das Update von Bludit v1.6 (getestet mit v1.6.1 und v1.6.2) auf v2 steht ein Migrations-Skript zur Verfügung:

<a href="https://github.com/anaggh/bludit-scripts/tree/master/migration" target="_blank">Migration Script for Bludit v1.6.2 to v2.0</a>

**Vorgehen**

1. Aus Sicherheitsgründen ein Backup der bestehenden Installation anlegen.
2. Die Datei migrate.php in das Verzeichnis mit der Installation von Bludit laden.
3. Das Skript ausführen, indem es im Browser aufgerufen wird (beispielsweise unter https://meinedomain.ch/migrate.php). Das Skript legt ein Verzeichnis `/migrates` an mit den aufdatierten Inhalten von `/bl-content`.
4. Bludit v2 auf den Server laden (nicht installieren!). Wenn das gleichen Verzeichnis wie für Bludit v1.6 verwendet wird, Bludit v1.6 deinstallieren, d. h. alle Dateien und Verzeichnisse mit Ausnahme des Verzeichnisses `/migrations`.
5. Den Inhalt des Verzeichnis `/bl-content` im Verzeichnis `/migrations` in das Verzeichnis `/bl-content` der neuen Bludit-Installation kopieren oder verschieben.

Wenn Inhalte aus irgendeinem Grund nicht migriert worden sind, befinden sich diese im Verzeichnis `/migrations/failed-posts`, von wo sie manuell übernommen werden können.

**Themes und Plugins**

Allenfalls muss das Theme angepasst werden.

Zudem müssen die neuen Versionen der Plugins installiert und aktiviert werden.

**Achtung: Nicht von allen Plugins sind aktualisierte Versionen verfügbar.**
